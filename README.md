# Sistema Integral Software FJ

## Descripción del proyecto

Este proyecto corresponde al desarrollo de la Fase 4 del curso de Programación de la Universidad Nacional Abierta y a Distancia (UNAD).

El sistema fue desarrollado utilizando programación orientada a objetos en Python y permite gestionar clientes, servicios y reservas para la empresa Software FJ.

La aplicación implementa manejo avanzado de excepciones, validaciones, herencia, polimorfismo, encapsulación y registro de eventos mediante archivos logs, garantizando estabilidad y robustez durante la ejecución.

---

## Funcionalidades principales

* Registro de clientes
* Gestión de servicios
* Creación y cancelación de reservas
* Manejo de excepciones personalizadas
* Registro de eventos y errores en logs.txt
* Simulación de operaciones válidas e inválidas
* Control de errores sin detener el sistema
